- **CLI 시작 다이얼로그**: 터미널 버튼 클릭 시 세션 없으면 팝오버 표시. 저장된 커맨드 선택지 + 커스텀 입력 + `+` 저장 버튼. `data/cli_patterns.json`의 `saved_commands` 활용
- **터미널 카드**: 컬럼 헤더의 작은 아이콘 대신, 스트림 맨 위에 전용 카드 배치. border 색상으로 상태 표시 (회색=off, 노랑=idle, 초록=running). ■ 클릭으로 세션 종료
- **Kill CLI 버튼**: 터미널 패널 헤더에 추가. 세션 종료 시 패널도 자동 닫힘
- **외부 CLI 감지 제거**: MPM tmux 세션만 인식 (mpm- prefix)
- **풀스크린 터미널 페이지**: `/terminal/<project>` — 별도 HTML
