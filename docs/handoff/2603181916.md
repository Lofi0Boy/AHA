### Commit 1 (2603181916) — 터미널 패널 초기 로드 시 플래시 제거

**문제**: 페이지 첫 로드 시 아무 터미널이 열려 있지 않아도 검은 터미널 패널이 잠깐 "슝" 하고 애니메이션되는 현상
- 원인: `applyTermMode()` 호출 시 CSS `transition: transform 0.25s ease`가 활성화된 상태에서 `transform` 값이 `translateY(100%)` → `translateX(100%)`로 바뀌며 애니메이션 발생
- 수정: 초기화 시 `termPanel.style.transition = 'none'`으로 트랜지션 비활성화 → 모드 적용 → reflow 강제 → 트랜지션 복원
